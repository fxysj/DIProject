下面是围绕你提出的“每天更懂你”个性化系统中 **UI（用户界面）设计师 / UX设计师** 的详细岗位能力要求，适用于平台类产品，如具备多模态输入、Agent对话、多工具流、多轮推荐等特点的智能系统：

---

## 🎨 UI / UX 设计师岗位要求

### 🌟 核心关键词：

> 个性化推荐、对话流体验、多模态输入、可视化工具链、用户情感链接

---

### 一、设计能力要求

#### ✅ 1. 对话交互体验设计

* 能设计出符合 Agent 对话节奏的多轮交互 UI（如 GPT-4 系统中不同步骤的 Tool/Agent可视化）
* 熟悉「问答+推荐」的混合式对话设计模式（例如：用户说“我想减肥”，系统给出计划 + 问题引导）
* 支持多步表单链式填写（Tool 参数采集流）

#### ✅ 2. 多模态输入与展示

* 了解图像/语音/视频/表格输入形式的 UX设计（如图文拖拽、语音识别进度）
* 能在 UI 上做输入状态可视化提示（如 Whisper 实时识别状态）

#### ✅ 3. 个性化推荐 UI/UX

* 会设计“千人千面”推荐信息流，如智能工具推荐栏、个性化模块
* 能基于用户画像生成“专属助手主页”概念草图（类似「为你定制」界面）

#### ✅ 4. Tool链调用可视化能力

* 能用图标 + 动画方式表达工具流（如“价格计算器已调用”，“地图定位工具已完成”）
* 熟悉流程型交互设计（Flow / Step by Step Tool 使用体验）

#### ✅ 5. Prompt 编辑器 / Agent Flow Builder 能力（加分）

* 能设计类 LangFlow、Flowise 等“节点+流向”型可视化编辑界面
* 支持 Tool 插件可插拔设计，支持 Agent 自定义图形搭建

---

### 二、体验优化意识

#### ✅ 6. Token 消耗 / 模型响应反馈体验优化

* 熟悉 LLM 上下文限制，可引导用户缩减输入
* 提供 Token 使用提示、响应超时处理、loading 动画等体验细节

#### ✅ 7. 用户反馈机制闭环设计

* 设计“你对这次推荐满意吗”“我有更详细的需求”等反馈组件
* 支持用户评价→反哺推荐系统→可视化结果回显（如“已优化你的偏好”）

---

### 三、审美与一致性要求

#### ✅ 8. 品牌统一视觉体系

* 统一色彩、图标、控件风格，打造智能产品一致性（避免「工具拼盘」式混乱）
* 具备 Web / App / 小程序 跨端设计能力

#### ✅ 9. 微交互动效能力（加分）

* 会使用 Figma + Principle / Framer / After Effects 制作高保真动画原型
* 会为关键交互节点添加细腻动效（如工具被激活、推荐卡片滑入等）

---

### 四、协作能力要求

#### ✅ 10. 与算法 / 前端的协作能力

* 能看懂 Tool 路由逻辑图，理解参数配置（Prompt 参数、Tool Slot 等）
* 可将交互文档结构化输出，便于前端快速实现（附状态流/空态/异常态设计）

---

## 🧩 UI/UX设计岗位能力总结表

| 模块            | 要求说明                          |
| ------------- | ----------------------------- |
| 多轮对话设计        | 有连续对话体验规划，能设计“引导式问答”与“推荐流”    |
| 多模态输入设计       | 会做图文/语音/视频输入交互组件              |
| 工具链视觉流程表达     | 会可视化 Tool 的调用、响应、结果显示流程       |
| 个性化内容展示       | 能做“为你推荐工具”“为你定制计划”的模块化 UI     |
| 用户反馈闭环设计      | 包含满意度反馈、主动补充输入等闭环组件           |
| Token 与响应体验设计 | 知道模型上下文长度约束，做合理提示与 loading 优化 |
| 设计协作能力        | 与前端/算法协同，交互流程结构清晰，支持敏捷交付      |
