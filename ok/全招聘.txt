# 🎯“每天更懂你”AI系统人才标准（六大岗位）

---

## 🧑‍💻 一、前端工程师（Web / 小程序 / App）

**关键词：多模态交互、可视化Tool链、Agent反馈闭环、低延迟响应**

### 核心能力要求：

1. **复杂对话 UI 构建经验**：能构建多轮多工具 Agent 的可视化对话体验（如 LangFlow 前端交互）
2. **多模态输入处理能力**：支持语音转写、图像上传、表格导入等复杂输入组件
3. **Token/上下文限制敏感设计**：对模型Token长度敏感，有过设计提示词动态预览/裁剪逻辑经验
4. **前端状态管理**：熟练掌握状态提升 + 缓存 + 前后端协同（例如一个步骤式多Agent系统状态）
5. **数据埋点闭环设计能力**：为用户行为设计可追踪的结构化数据流（支持后续行为建模）

> **加分项**：有 LangChain UI / Prompt UI / 可视化 Flow Builder 经验

---

## 🧑‍🔧 二、后端工程师（Golang / Python / Java）

**关键词：Tool调用链路、LangGraph调度、状态机、多模态预处理、缓存优化、模型压缩部署**

### 核心能力要求：

1. **熟悉 LangChain / LangGraph / 自研工具链调度框架**
2. **Agent 调用系统构建能力**：支持Tool注册、调用状态流转、失败兜底等机制
3. **多模态预处理与中间缓存层设计**：例如 Whisper 转写语音、图片转base64缓存等
4. **私有部署 / 推理服务编排能力**：具备使用 FastAPI、vLLM、MLC、Triton 等部署经验
5. **性能优化意识强**：对 Token 消耗、上下文截断、历史缓存、Async 调用有优化经验
6. **用户状态持久化与回溯能力**：如基于 Redis、MySQL 构建对话历史/多步骤上下文恢复机制

> **加分项**：具备部署过私有 LLM / Whisper / 多模态模型的经验

---

## 🧠 三、算法工程师 / 大模型工程师

**关键词：个性化推荐、用户行为建模、意图识别、多模态理解、模型压缩与落地**

### 核心能力要求：

1. **用户行为建模能力**：包括兴趣标签建模、Embedding演化、序列建模（Transformer/RNN）
2. **意图识别与Tool Routing能力**：支持多轮意图追踪与slot填充（例如“我想给他转300块”）
3. **多模态理解能力**：有图文/语音/视频意图提取与对齐经验（如 CLIP/BLIP/Whisper）
4. **推荐算法经验**：擅长用向量检索、用户Embedding聚类、召回/重排序方法
5. **模型压缩 / 蒸馏 / 量化经验**：如 QLoRA、GGML、MLC、BGE-Small、Whisper-Tiny 等轻量模型处理经验

> **加分项**：熟悉 LangChain Agent设计思想 + HuggingFace + RAG增强检索系统

---

## 🧪 四、测试工程师 / QA工程师

**关键词：AI工具系统稳定性、Tool链可追踪性、多模态输入测试、行为路径回溯**

### 核心能力要求：

1. **熟悉 AI 多轮对话类系统测试方法**：包括上下文连续性、Tool调用正确性、fallback逻辑
2. **具备多模态测试经验**：测试语音转写正确性、图片处理边界、文本内容审查
3. **行为预测链路可回溯性验证**：能跟踪用户输入 → 意图识别 → Tool调用 → 响应生成 全链条日志
4. **自动化测试能力**：能搭建用于 LLM 工具链的 Mock Server + Coverage Tracker
5. **Token/成本可控测试**：测试系统Token消耗点，防止无效调用或提示词超长等情况

> **加分项**：懂 prompt injection / jailbreak 风控场景测试

---

## 🧑‍💼 五、产品经理

**关键词：以用户为中心的工具推荐系统设计、AI闭环、用户生命周期管理、A/B实验**

### 核心能力要求：

1. **具备构建“个性化助手”类产品经验**：能画出工具链 + 用户行为 + 模型推荐的产品闭环图
2. **多Agent流程协作意识强**：能梳理从意图识别 → 参数提取 → 工具调用 → 用户反馈的完整路径
3. **熟悉推荐系统或用户画像系统设计**：如“理财工具推荐系统”“知识工具推荐系统”等
4. **数据驱动意识强**：能根据日志 / 埋点指标定义“用户是否获得了精准推荐”的度量方式
5. **与算法/后端高效协同能力**：能描述 prompt、模型需求、参数格式规范、输入输出标准等

> **加分项**：主导过大模型驱动的 Agent / 工具类产品，有 LangChain、AgentFlow 产品结构理解

---

## 📣 六、运营 / 用户增长 / AI社群生态方向

**关键词：用户反馈闭环、提示词调优、场景引导、冷启动、AIGC内容驱动运营**

### 核心能力要求：

1. **具备AIGC工具类产品冷启动运营经验**：熟悉早期用户运营、互动任务设计、使用引导
2. **提示词调优能力**：能通过用户反馈调整 prompt 模版，引导更自然的交互
3. **内容场景构建能力**：设计“早上问候”“情绪关怀”“智能理财建议”等场景驱动场景运营
4. **增长数据敏感**：熟悉留存、转化、分享、成本等增长指标 + A/B 测试能力
5. **运营AI社群经验**：可推动用户共创 prompt、工具链、反馈循环

> **加分项**：能与产品 / 算法协作定义提示词模板、使用路径、Tool 调用反馈表单
